Prueba Técnica - Analista programador

Utilizando la API de JsonPlaceHolder (https://jsonplaceholder.typicode.com) se requiere que se realice un portal de administrador, este portal debe incluir una sección de usuarios (/users) en donde se debe listar los usuarios y poder crear nuevos usuarios, también se requiere que tenga una sección de fotos (/photos), en donde debe mostrarse el directorio de fotos que contiene la API en una card cada una.



Requisitos de la prueba:

  • Debe crear un repositorio en Github o Bitbucket, subir el proyecto y dar permiso de admin a "raul.jaque@dimerc.cl"
  • Se puede utilizar cualquier tecnología de su preferencia para el desarrollo del Frontnd de la aplicación y la lógica de las APIS, se recomienda el uso de Bootstrap para los estilos
